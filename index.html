Project 2


Description:

The To-Do List Application is a simple, user-friendly tool designed to help users manage tasks efficiently. This web-based application enables users to create, view, edit, and delete tasks, providing a clear and organized way to keep track of daily responsibilities and goals. Each task can be categorized by priority or due date, helping users prioritize their workload and enhance productivity.

Key Features:

  • Add New Tasks: Allows users to add tasks with specific details like title, description, priority, and due date.
  • Mark Tasks as Completed: Users can mark tasks as completed, providing a sense of progress.
  • Edit and Delete Options: Tasks can be edited if details change or deleted when no longer needed.
  • Task Filtering and Sorting: Enables filtering tasks by completion status or priority level for easy navigation.
  • User-Friendly Interface: Simple, intuitive design for easy navigation and use on both desktop and mobile devices.

Technologies Used:

HTML, CSS, JavaScript

Objective

The To-Do List Application aims to simplify task management, helping users improve time management and productivity by keeping track of their goals and responsibilities in a structured format. It’s a valuable tool for students, professionals, or anyone who needs a simple, effective way to organize daily tasks.
